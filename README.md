# Gulp使用心得和搭建手册

### 安装Gulp
	命令: npm install --global gulp
	(淘宝镜像)cnpm i -S gulp

	###### (淘宝镜像的安装)
	命令:  npm install -g cnpm --registry=https://registry.npm.taobao.org
